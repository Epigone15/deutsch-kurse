# Deutsch-Kurse : Apprendre les cas allemands et les adjectifs

Bienvenue sur **Deutsch-Kurse**, un site web éducatif conçu pour aider les francophones à maîtriser les **4 cas allemands** (Nominatif, Accusatif, Datif, Génitif) et la **déclinaison des adjectifs**. 

## 🌍 Objectif
Ce projet vise à rendre l'apprentissage de la grammaire allemande **accessible, interactif et ludique** grâce à :
- Des explications claires en français
- Des tableaux de déclinaison complets
- Des exercices interactifs avec correction instantanée
- Un tuteur IA pour répondre à tes questions

## 🛠 Technologies utilisées
- **HTML5** : Structure du site
- **CSS3** : Design responsive et moderne
- **JavaScript** : Logique des exercices et simulation d'IA

## 🚀 Comment utiliser ce projet ?
1. Clone ce dépôt :
   ```bash
   git clone https://github.com/Epigone15/deutsch-kurse.git
   ```
2. Ouvre `index.html` dans ton navigateur.
3. Explore les sections et entraîne-toi !

## 📂 Structure des fichiers
- `index.html` : Page principale
- `style.css` : Styles CSS
- `script.js` : Logique JavaScript (exercices, IA simulée)
